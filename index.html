À propos

La caméra du pack sécurité de la Freebox Delta (F-HACAM01A) se configure normalement via l'application Freebox. Ce générateur crée le QR Code de provisionnement directement, ce qui vous permet de connecter la caméra à n'importe quel routeur WiFi WPA2 — sans Freebox Delta, sans compte Free.

Le QR Code encode le SSID, le mot de passe WiFi et le nom de la caméra en clair. Aucun compte Free n'est requis. Fonctionne avec tout routeur WPA2.

Utilisation du QR Code

Étapes de provisionnement

  1. Mettez la caméra sous tension. Attendez que la LED clignote lentement en orange — la caméra est en mode provisionnement (paired=0).
  2. Présentez le QR Code face à l'objectif de la caméra, à environ 20–40 cm. La caméra capture des images en continu et décode le QR via zbarimg.
  3. Attendez la LED verte fixe — la caméra a lu le QR, s'est connectée au WiFi et démarre le flux vidéo.
  4. Trouvez l'adresse IP attribuée par votre routeur (via l'interface DHCP ou un scan réseau : nmap -sn 192.168.1.0/24). Le hostname DHCP sera celui que vous avez saisi.

Met à jour les URLs des flux ci-dessous en temps réel

  1. Accédez aux flux avec les identifiants : freeboxcam / voir mot de passe calculé ci-dessus.

État LEDSignification

URLs des flux vidéo

Remplacez <IP> par l'adresse de votre caméra :

  • HLS : http://freeboxcam:<MOT_DE_PASSE>@<IP>/img/stream.m3u8
  • Snapshot JPEG : http://freeboxcam:<MOT_DE_PASSE>@<IP>/img/snapshot.cgi
  • RTSP HD : rtsp://freeboxcam:<MOT_DE_PASSE>@<IP>:554/live.sdp
  • RTSP SD : rtsp://freeboxcam:<MOT_DE_PASSE>@<IP>:554/live2.sdp
⚠️ Activation requise avant le premier accès RTSP
Les chemins RTSP (sp_uri, sp_uri2) sont vides par défaut après provisionnement. Sans cette étape, VLC et les autres lecteurs retourneront une erreur 404.

Exécutez cette commande une seule fois : 
curl -u freeboxcam:<MOT_DE_PASSE> "http://<IP>/adm/set_group.cgi?group=H264&sp_uri=live.sdp&sp_uri2=live2.sdp"
Les flux RTSP deviennent alors accessibles. Générez le QR Code et renseignez l'adresse IP ci-dessus pour obtenir les URLs complètes. Documentation complète →
Re-provisionnement : Si vous devez changer de réseau WiFi, effectuez un reset usine via http://<IP>/adm/reset_to_default.cgi (avec authentification) ou maintenez le bouton physique 10 secondes. La caméra repassera en mode provisionnement.
Compatibilité : Cette procédure fonctionne avec tout routeur WiFi WPA2 (2.4 GHz). La caméra ne supporte pas le WPA3 ni le WiFi 5 GHz.
